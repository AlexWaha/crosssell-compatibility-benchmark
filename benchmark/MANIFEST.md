# AVTC Curated Benchmark Dataset (v2)

Controlled cross-sell benchmark derived from the full 14767-product catalog.
Built to fix the failed open-catalog experiment: the full catalog had no ground
truth and semantic retrieval surfaced similar (not complementary) products, so
Precision/Recall/NDCG could not be measured. This subset has explicit, derivable
ground truth and hard negatives.

Generated by `build.py` (deterministic, `SEED=42`). Re-run to reproduce.

## Composition

| Layer | Categories | Products |
|-------|-----------|----------|
| Anchors (devices with upsell) | 12 | 360 |
| Accessories (compatible by graph) | 32 | 959 |
| **Core (real pairs)** | 44 | **1319** |
| Noise / distractors (must never be recommended) | - | 1681 |
| **Total** | | **3000** |

Ground-truth positive pairs: **47670** (anchor x accessory, many-to-many).

## Verticals

1. **Mobile** - Mobile Phones, Tablets -> cases, chargers, power banks, headsets, memory cards
2. **Photo** - Digital Cameras, Camcorders, Action Cameras -> lenses, filters, tripods, bags, batteries, flashes, memory cards
3. **Computing** - Laptops, Desktop PCs, Monitors -> bags, mice, keyboards, pads, RAM, SSD, GPU, motherboards
4. **Cycling + selective auto/tools** - Bikes -> lights, computers, locks, tyres, bottles; Drills -> bits/kits/boxes; Welders -> helmets; E-Motorbikes -> helmets, tyres

## Ground truth model

A pair `(anchor, accessory)` is **positive** iff the accessory product belongs to
an accessory category mapped to one of the anchor product's anchor categories in
`ground_truth_graph.json`. Many-to-many is intentional and realistic (a memory
card pairs with phones, cameras, and laptops). Noise products appear in **no**
positive pair - they are the precision test.

This is **category-level** ground truth: coarse but reproducible, LLM-independent,
and free of human labeling drift. The LLM verifier is evaluated against it, so the
verifier is never part of its own ground truth.

## Files

| File | What |
|------|------|
| `products.json` | 3000 products, fields: `id, name, description, brand, category_ids, specs, image_path, price` |
| `categories.json` | Used categories + ancestor closure |
| `attributes.json` | Global attribute lookup (group_id -> attr_id -> name), copied whole |
| `ground_truth_pairs.json` | Expanded `[{anchor_id, accessory_id, via_category}]` (47670) |
| `ground_truth_graph.json` | Human-readable `anchor_cat -> accessory_cats` map |
| `roles.json` | `product_id -> anchor | accessory | noise` |
| `counts.json` | Control numbers (below) |
| `images/<NNNN>/<FILE>.jpg` | 3000 product images (~330M), mirrors `image_path` |
